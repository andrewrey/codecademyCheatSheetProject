HTML Cheat Sheet

A reference sheet for users learning HTML, showing the most common tags used by developers

Tag Name Description
<html> HTML Tag The HTML tag is used as the root of the document which specifies that the document is html
<body> Body Tag The body tag is used to define the body of html document. It contains the images, tables, lists, and more.
<title> Title Tag The title tag is used to define the title of the html document
<h1> - <h6> Heading Tag The heading tag is used to define the heading of the html document.
